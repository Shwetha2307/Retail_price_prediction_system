# 💰 Retail Price Prediction System

## 🔍 Overview

Retail Price Intelligence System is an AI-powered web application that analyzes and predicts product prices across multiple e-commerce platforms like Amazon and Flipkart.

It helps users:

* Compare real-time prices
* Identify the best platform to buy
* Predict future price trends using Machine Learning
* Make smart BUY/WAIT decisions

---

## 🚀 Key Features

### 📊 Price Comparison

* Compares product prices across Amazon and Flipkart
* Identifies the cheapest platform instantly

### 📉 Price Trend Visualization

* Displays historical price trends using graphs
* Helps understand market behavior over time

### 🔮 AI Price Prediction

* Uses Linear Regression to predict future prices
* Forecasts next-day price based on historical data

### 🧠 Smart Recommendation System

* Provides BUY or WAIT suggestion based on prediction
* Helps users save money intelligently

### 📈 Price Difference Analysis

* Calculates percentage difference between platforms
* Highlights savings clearly

### ⚡ Volatility Analysis

* Measures price stability using standard deviation
* Helps understand market fluctuations

## 📊 How It Works

1. User selects a product
2. System retrieves historical price data
3. Compares Amazon vs Flipkart prices
4. Trains Linear Regression model on price trends
5. Predicts next-day price
6. Provides recommendation (BUY / WAIT)
